# Conditionals in JavaScript

Conditional statements are used to make decisions in a program.

## if Statement

```javascript
if (condition) {
    // code
}
```

## if...else Statement

```javascript
if (condition) {
    // code
} else {
    // code
}
```

## else if Statement

```javascript
if (condition1) {

} else if (condition2) {

} else {

}
```

## switch Statement

```javascript
switch(expression) {
    case value:
        // code
        break;

    default:
        // code
}
```
