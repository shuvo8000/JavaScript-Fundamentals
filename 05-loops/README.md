# Loops in JavaScript

Loops are used to execute a block of code repeatedly.

## for Loop

```javascript
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```

## while Loop

```javascript
while (condition) {
    // code
}
```

## do...while Loop

```javascript
do {
    // code
} while (condition);
```

## break Statement

Stops the loop immediately.

```javascript
break;
```

## continue Statement

Skips the current iteration.

```javascript
continue;
```
