# Data Types in JavaScript

JavaScript provides different types of data.

## Primitive Data Types

1. String
2. Number
3. Boolean
4. Undefined
5. Null
6. BigInt
7. Symbol

## Non-Primitive Data Types

1. Object
2. Array
3. Function

## Example

```javascript
let name = "Shuvo";
let age = 23;
let isStudent = true;
```

## typeof Operator

Used to check the type of a variable.

```javascript
console.log(typeof name);
```
