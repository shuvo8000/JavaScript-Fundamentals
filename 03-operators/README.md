# Operators in JavaScript

Operators are used to perform operations on values and variables.

## Types of Operators

### Arithmetic Operators

| Operator | Meaning |
|----------|----------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |
| ** | Exponentiation |

### Assignment Operators

```javascript
x += 5;
x -= 5;
x *= 5;
```

### Comparison Operators

```javascript
==
===
!=
>
<
>=
<=
```

### Logical Operators

```javascript
&&
||
!
```

## Important

- `==` checks only value.
- `===` checks value and data type.
